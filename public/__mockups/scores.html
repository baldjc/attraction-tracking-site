Design notes
  • Threshold-based colour scaling on the score circle — emerald at ≥7.5, amber at ≥5, crimson below 5. The same colour cascade applies to the principle bars and the line-chart end dot, so a member's eye learns the colour grammar in one glance.
  • The Learning Path is the page's only azure-tinted background — it's the page's single CTA. Everything else is white-on-warm; the azure tint makes the next action unmissable without screaming.
  • All scores render in Geist Mono with tabular-nums so the principle table lines up perfectly down the column and 6.8 vs 6.4 reads as a clean delta instead of optical jitter.
  • Line chart shows the actual trajectory — flat would be boring, this climbs from 5.2 to 6.8 over 12 weeks with realistic variance. The last point is highlighted emerald to anchor the "you're trending up" message.
  • Recent audits use a horizontal scroller, not a grid — keeps the page short and signals "there's more here if you want it" without committing to vertical real estate.
