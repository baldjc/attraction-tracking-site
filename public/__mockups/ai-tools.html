Workflow

The order matters.

Five tools, used in sequence, become muscle memory for shipping a channel that converts. Follow the order until you can run it from memory.

  1. Start with Avatar Architect to define exactly who you’re talking to. Once per quarter
  2. Run Content Engine weekly to generate validated ideas from your data. Weekly
  3. Use ARC Script Builder to write the script, with cited facts auto-anchored. Per video
  4. Score with Title & Thumbnail Analyzer before you publish. Per video
  5. Run Script Review for a second-pass quality check. Per video
Design notes
  • The 64×64 sky-tinted icon blocks use the AI Tools feature colour (#6BA3C7 at 10% opacity background, full saturation icon). All five cards share the colour because this is the index, not the individual tools — the per-tool colour identity comes later, inside each tool page.
  • The recent activity badge prevents the page from reading as static. "Last edited 3 days ago" / "12 ideas this week" / "5 drafts in progress" communicates "I’m a tool that I’ve actually used." Without it the page looks like a marketing brochure.
  • The workflow block at the bottom doubles as in-app onboarding. First-time members learn the order; returning members get a glanceable reminder. Frequency tags (once per quarter, weekly, per video) tell them what to plan around.
  • One azure moment per region: the eyebrow pill at the top and the “matters” word in the workflow heading. Cards stay quiet on warm off-white, so the tools themselves don’t fight each other for attention.
  • The 5th card centers below the 2×2 grid rather than orphaning in a third column — visually pleasant, and reinforces Script Review’s position as the final pass.
