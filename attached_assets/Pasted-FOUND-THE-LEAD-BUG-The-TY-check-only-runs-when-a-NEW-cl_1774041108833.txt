  guides.chamberlaingroup.ca/9?ref=xxx
  2. Click endpoint: page_url="/9", saved="/thank-you" →    
  match=false (correct, this isn't the TY page)             
  3. New click created ✓
  4. User fills form → redirected to                        
  www.chamberlaingroup.ca/thank-you/?ref=xxx
  5. Click endpoint: same ref+IP within 30 seconds → DEDUP, 
  returns existing click                                    
  6. NO TY CHECK HAPPENS on the dedup path → lead never
  created

  current page_url matches the thank you page. When a dedup
  happens:                                                  
  - Still return the existing click (don't create a new one)

  existing click

  exists for this click_id
