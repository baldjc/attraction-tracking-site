  Add a "Test Installation" section on the Link Tracking Setup page, below the snippet.

  How it works:
  1. "Test Installation" button
  2. When clicked:
     a. Generate a temporary test ref code like "test_" + 8 random chars
     b. Create a temporary test TrackingLink for this member (flagged somehow as test data)
     c. Show the member: "Open this link in a new tab:" with a clickable link to their
  destination URL + ?ref={testRefCode} (opens in new tab)
     d. Start polling GET /api/tracking/verify?refCode={testRefCode} every 2 seconds for 30
  seconds
     e. The verify endpoint checks if a Click record exists for that ref code

  3. Results:
     - SUCCESS (click found): Green checkmark — "Your snippet is working! We detected your
  test visit." Then clean up: delete the test TrackingLink and test Click data so it doesn't

       * "Make sure the tracking snippet is pasted inside the <head> tag on every page of
  your site"
       * "Check that the data-id in the snippet matches your member ID"
       * "Disable ad blockers and try again"
       * "The snippet must be on the page you're testing"
     - "Try Again" button to restart

  4. API: GET /api/tracking/verify?refCode={refCode} — returns { found: true/false }
