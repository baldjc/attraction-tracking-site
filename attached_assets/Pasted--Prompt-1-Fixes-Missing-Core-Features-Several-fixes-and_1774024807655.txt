  Prompt 1: Fixes + Missing Core Features

  Several fixes and missing features for the campaigns & tracking links system. Do all of
  these:

  === FIX 1: INPUT TEXT COLOUR ===
  The text typed into input fields on the New Campaign and New Tracking Link modals is too
  light gray and hard to read. The placeholder text is fine being light, but the actual
  typed VALUE text must be dark (text-gray-900 or similar). Fix on all inputs and select
  dropdowns in both modals.

  === FIX 2: data-ty MISSING FROM TRACKING SNIPPET ===
  On the Link Tracking Setup page, the tracking snippet preview is missing the data-ty
  attribute. When the member saves their Thank You Page Path (e.g., /thank-you), the snippet
   must include it:

  <script src="https://members.attractionbyvideo.com/api/t.js" data-id="{memberId}"
  data-ty="{thankYouPagePath}" defer></script>

  Right now it only shows data-id. Without data-ty, the JavaScript snippet doesn't know
  which page counts as a lead conversion, so no leads will ever be recorded.

  - When thank you page path is saved, regenerate the snippet preview to include data-ty
  - "Copy snippet" must copy the version WITH data-ty
  - If no thank you path is saved yet, omit data-ty but show a warning: "Save your thank you
   page path above to enable lead tracking"

  Also verify that t.js actually reads the data-ty attribute and uses it to detect the thank
   you page. The logic should be: if window.location.pathname matches the data-ty value,
  fire the lead conversion endpoint.

  === FIX 3: DYNAMIC LABEL BASED ON SOURCE TYPE ===
  When the campaign's source type is YouTube, the New Tracking Link modal should say "Video
  Name" instead of "Link Name" (label and placeholder). For other source types keep "Link
  Name".

  === FIX 4: EDIT TRACKING LINK AFTER CREATION ===
  Add an edit button (pencil icon or "..." menu) on each tracking link row. Clicking opens a
   modal to update:
  - The link name / video name
  - The YouTube Video URL (can be added later if left blank at creation)

  API: PATCH /api/campaigns/[id]/links/[linkId]
  If YouTube URL is added/changed, extract and save the video ID too.

  === FIX 5: DELETE CAMPAIGNS AND LINKS ===
  Add delete functionality:
  - Campaign cards: "..." menu or trash icon → confirmation dialog → soft delete (set
  deletedAt = NOW())
  - Also soft delete all child TrackingLinks when campaign is deleted
  - Individual tracking link rows: delete option → soft delete
  - NEVER delete Click, PageView, or Lead records
  - Filter out soft-deleted records from all list queries (WHERE deletedAt IS NULL)
  - Confirmation dialog with red Delete button: "Delete [name]? Historical click and lead
  data will be preserved."

  API:
  - DELETE /api/campaigns/[id]
  - DELETE /api/campaigns/[id]/links/[linkId]
