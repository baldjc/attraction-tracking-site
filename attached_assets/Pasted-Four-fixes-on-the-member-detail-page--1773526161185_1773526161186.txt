Four fixes on the member detail page:                                                                                                  
                                                                                                                                         
  1. **Membership Level needs to be a dropdown** — Right now "Membership Level" shows as plain text ("Foundations"). Make it an editable 
  dropdown when in edit mode with these options: Foundations, Editing 2, Editing 4, Mastery 2, Mastery 4. Colour-code the badge in the   
  header and the membership level display:                  
     - Foundations → blue (#3dc3ff)
     - Editing 2 / Editing 4 → amber/orange
     - Mastery 2 / Mastery 4 → purple
     Apply this colour coding to the badge in the header banner AND the value in Member Info.

  2. **YouTube channel name instead of raw handle** — When the YouTube handle is a channel ID like "@UC8cdI58o5qIfOeAw2TQ4xYA" instead of
   a readable handle, it looks bad. During GHL sync and on the member detail page, use the YouTube Data API to look up the channel and
  get the actual channel name (snippet.title). Store this as a new field "youtube_channel_name" on the user. Display the channel name in
  the header banner and Member Info instead of the raw handle/ID. If a readable handle exists (like @chamberlaingroup), show both:
  "Chamberlain Group (@chamberlaingroup)". If it's just a channel ID, show only the channel name.

  3. **Phone number from GHL** — Add a "phone" field to the users table. During GHL sync, pull the contact's phone number from the GHL
  contact data (it's in the "phone" field on the contact object). Display it in the Member Info section between Email and YouTube URL.
  Also add a "Call Member" quick action button that uses tel: link.

  4. **Update the GHL sync to pull phone and YouTube channel name** — When syncing contacts from GHL, for each contact:
     - Save their phone number if available
     - If they have a YouTube channel URL/ID, call the YouTube Data API to get the channel name and save it
     - Only call the YouTube API for contacts that don't already have a channel name saved (to avoid unnecessary API calls)
