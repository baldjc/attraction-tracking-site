Three changes to the campaigns/tracking links feature:                                    
                                                                                            
  1. **Fix input text colour on all modals** — The text typed into input fields (Campaign   
  Name, Destination URL, Link Name, Video URL) is too light/gray and hard to read. The      
  *placeholder* text is fine being light, but the actual typed value text needs to be dark  
  (text-gray-900 or text-[#1e2a38]). Same for the select dropdown. Fix this on both the New
  Campaign modal and the New Tracking Link modal.

  2. **Dynamic label based on source type** — When the campaign's source type is YouTube,   
  the "New Tracking Link" modal should say "Video Name" instead of "Link Name" (label and
  placeholder). For all other source types (Google Ads, Email, Other), keep it as "Link     
  Name". The campaign detail page already knows the sourceType, so pass it to the modal.  

  3. **Allow editing a tracking link's YouTube URL after creation** — Add an edit button    
  (pencil icon or "..." menu) on each tracking link row in the campaign detail page.
  Clicking it opens a modal or inline edit that lets the user update:                       
     - The link name / video name                                                         
     - The YouTube Video URL (add it later if it was left blank at creation)                
                                                                                            
     This is important because members will often create the tracking link BEFORE they've   
  filmed and uploaded the video. They need to come back and attach the YouTube URL later.   
                                                                                            
     Use a PATCH endpoint like PATCH /api/campaigns/[id]/links/[linkId] to update these     
  fields. If the YouTube URL is added/changed, extract the video ID from the URL and update
  youtubeVideoId too.
