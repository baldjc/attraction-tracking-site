Build a member Dashboard page. This should be the default 
  landing page when a member logs in (replace whatever is
  currently at /member or /member/dashboard). It should show
   a motivating, actionable snapshot of their YouTube
  progress.                                                 
                                                            
  === LAYOUT ===                                            
                                                            
  The page title is "Dashboard" with a greeting: "Welcome   
  back, {firstName}" (or just their first name).            
                                                            
  === ROW 1: KEY METRICS (4 cards across the top) ===       
                                                 
  1. **Attraction Score** — Big number showing their current
   score (from latest audit). Show a trend arrow: green ↑ if
   higher than previous audit, red ↓ if lower, grey → if    
  same. Below the number show "from {audit date}" in small
  text.                                          

  2. **Leads This Month** — Total leads from their campaigns
   this calendar month. Below show comparison: "+X vs last
  month" in green or "-X vs last month" in red.             
                  
  3. **Clicks This Month** — Same treatment as leads.

  4. **Conversion Rate** — Current month's conversion rate  
  (leads ÷ clicks). Trend vs last month.
                                                            
  If a member has no campaigns or tracking data yet, show
  "0" with a link: "Set up link tracking →" pointing to Link
   Tracking Settings.

  If a member has no audits yet, show "—" for Attraction    
  Score with text: "Awaiting first audit"
                                                            
  === ROW 2: TWO COLUMNS ===
                                                 
  LEFT COLUMN (wider, ~60%):

  **Top 3 Strengths & Top 3 Gaps**
  Pull from their latest audit's 16-principle breakdown.
  - Show 3 highest-scoring principles with green badges and 
  their scores
  - Show 3 lowest-scoring principles with red/orange badges 
  and their scores                                          
  - Below the gaps, show a smart suggestion for each gap:
  map the weak principle to an AI tool they can use:        
    - Low Title Frameworks → "Try the Title Creator →" (link
   to AI Tools)                                             
    - Low Avatar Clarity → "Build your Avatar →" (link to
  Avatar Architect)                                         
    - Low ARC Attention/Revelation/Connection → "Outline a
  script with ARC Script Builder →"                         
    - Low Lead Magnet → "Set up campaign tracking →" (link
  to Campaigns)                                             
    - For other principles, suggest: "Review this in your
  latest audit report →" (link to the report)               
                  
  **One-Sentence Diagnosis**                     
  Below the strengths/gaps, show the one-sentence diagnosis
  from their latest audit in a quote-style block. Italic    
  text, subtle background.
                                                            
  RIGHT COLUMN (~40%):
                                                 
  **Days Since Last Upload**
  A simple counter showing how many days since their last
  YouTube upload (pull from their most recent video data if 
  available, or from audit data showing upload dates).
  - 0-7 days: green with checkmark "You're on track!"       
  - 8-14 days: yellow with warning "Time for a new video"
  - 15+ days: red with alert "Your consistency score will
  drop"                                                     
  - If no upload data available, show "Connect a YouTube
  video to start tracking" with link to Campaigns           
                  
  **Next Coaching Call**                                    
  Show "Next Q&A Call" with the date of the next upcoming
  Thursday. Format: "Thursday, March 27" with a relative    
  label like "in 4 days". Always show the next Thursday from
   today's date — these calls happen every Thursday.        
                  
  **Quick Links**                                
  A small grid of icon buttons linking to the AI tools they
  might want to use:
  - Title Creator
  - Script Builder
  - Script Review                                           
  - Title & Thumbnail Analyzer
  Label it "AI Tools" with a "See all →" link.              
                  
  === ROW 3: RECENT ACTIVITY ===                            
                  
  **Best Performing Video This Month**           
  If they have tracking links with YouTube videos, show the
  top-performing one:                                       
  - Video thumbnail + title
  - Clicks, Leads, Conv. Rate for this month                
  - "View campaign →" link                                  
                                                 
  **Score Trend** (compact sparkline)                       
  A small line chart showing their Attraction Score over the
   last 6 months (or however many audits they have). Compact
   version — not the full My Scores page chart. Label:
  "Score History" with "See details →" linking to My Scores.
                  
  === RESPONSIVE ===                             
  - On mobile, stack everything single-column
  - Cards should be concise — no wasted space
  - The dashboard should load fast — use parallel API calls 
  for the different data sources                            
                                                            
  === API ===                                               
  Create a single endpoint that returns all dashboard data
  in one call:                                              
                  
  GET /api/member/dashboard                                 
  Returns:
  - latestAudit: score, date, strengths (top 3), gaps       
  (bottom 3), oneSentenceDiagnosis                          
  - previousAudit: score (for trend comparison)  
  - campaignStats: { thisMonth: { clicks, leads, convRate },
   lastMonth: { clicks, leads, convRate } }
  - bestVideo: { title, thumbnail, clicks, leads, convRate,
  campaignId }                                              
  - daysSinceUpload: number or null
  - nextCoachingCall: next Thursday date                    
  - scoreHistory: array of { date, score } for sparkline    
                                                            
  This endpoint returns only the logged-in member's data.
