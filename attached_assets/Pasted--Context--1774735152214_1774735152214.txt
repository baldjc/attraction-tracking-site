  Context                                                                                                                                                    
                                                                                                                                                             
  The admin currently has two separate pages for member data that overlap significantly:                                                                     
                                                                                                                                                             
  - Foundations Members (/admin/members) — member directory with name, email, YouTube, tier, score. Click into a member for profile editing, coaching notes, 
  avatar, audit history, academy progress, campaigns.                                                                                                        
  - Member Analytics (/admin/analytics) — engagement dashboard with KPI cards, recent videos grid, and a member table with 7-day activity stats (videos, tool
   uses, clicks, conversions, active/at-risk/inactive status).                                                                                               
   
  These need to be merged into one unified Members page.
